# عملگرهای ساده

شما می‌توانید عملیات ریاضی را با عملگر‌های ساده‌ی زیر انجام دهید:

* **جمع**: ```c = a + b```
* **تفریق**: ```c = a - b```
* **ضرب**: ```c = a * b```
* **تقسیم**: ```c = a / b```

همچنین شما می‌توانید مانند ریاضیات از پرانتز‌ها برای تعیین اولویت و جدا کردن گروهی از عملگرها استفاده کنید: ```c = (a / b) + d```

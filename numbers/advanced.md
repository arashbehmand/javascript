# عملگر‌های پیشرفته

بعضی عملگر‌های پیشرفته‌تر نیز برای استفاده در اختیار شما هست:

* **پیمانه (باقیمانده تقسیم)**: ```x = y % 2```
* **افزایش**: با فرض a = 5
    * ```c = a++```, خواهیم داشت: c = 5 و a = 6
    * ```c = ++a```, خواهیم داشت: c = 6 و a = 6
* **کاهش**: با فرض a = 5
    * ```c = a--```, خواهیم داشت: c = 5 و a = 4
    * ```c = --a```, خواهیم داشت: c = 4 و a = 4
